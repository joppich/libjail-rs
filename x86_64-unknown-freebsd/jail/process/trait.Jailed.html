Required Methods

fn jail(&mut self, jid: i32) -> &mut Command

Sets the child process to be executed within a jail. This translates to calling jail_attach in the child process. Failure in the jail_attach call will cause the spawn to fail.

Examples

use std::process::Command;
use jail::sys::jail_getid;
use jail::process::Jailed;

let output = Command::new("hostname")
             .jail(jail_getid("testjail").unwrap())
             .output()
             .expect("Failed to execute command");

println!("output: {:?}", output.stdout);
